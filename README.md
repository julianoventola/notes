# Features

- Main page showing all notes
- Add page to add a new note
- Edit page (clicking on note), to edit it
- Edit page (clicking on note), to delete it

## Getting Started

- Clone the repository
- run in cmd:
  ```cmd
  flutter pub get
  flutter run
  ```
